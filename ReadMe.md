# My Theresa (CLONE)

## Run Locally

Clone the project

```bash
  git clone https://github.com/Abrar2003/homely-wilderness-2173.git
```

Go to the project directory

```bash
  cd homely-wilderness-2173
```

Install dependencies

```bash
  npm install
```

Start the localhost server

```bash
  npm start
```
